## English - User Guide

ComfyPanel is a professional AI-native collaboration plugin for Adobe Photoshop. It transforms complex AI workflows into a streamlined, high-performance panel directly within PS, bridging local creative power with scalable cloud intelligence.

### 🌟 Key Features

- **Multi-Engine Strategy**: Seamlessly switch between **Local ComfyUI**, **NanoBanana**, and high-performance **Cloud backends** (e.g., BizyAir, RunningHub).
- **Professional App Management**:
  - **Modular Architecture**: Normalizes complex nodes into clean, professional UI widgets.
  - **Multi-Account & Sync**: Save multiple API keys and automatically sync private apps/workflows.
- **Deep Canvas Integration**:
  - **Real-time Awareness**: Responds to layer visibility, selections, and transformations instantly.
  - **Strategic Locking**: Freeze specific node previews to manage multi-reference workflows independently.
  - **Direct Flow**: Import results directly to layers or selection masks without black-flashes or lag.
- **Designer-First UI**: A premium, minimalist interface designed to feel like a native part of Photoshop.

### 💻 System Requirements

- **Photoshop Version**: Adobe Photoshop 2025 (v26.0) or higher.

### 🛠️ Installation

1. **Download**: Obtain the `ComfyPanel` plugin folder or `.ccx` installer.
2. **Installation**:
   - **Auto (Recommended)**: Double-click `comfypanel.ccx` and follow the Adobe installer prompts.
   - **Manual (UXP Directory)**:
     - **macOS**: Copy the `photoshop_plugin/ComfyPanel` folder to `~/Library/Application Support/Adobe/UXP/Plugins`.
     - **Windows**: Copy the folder to `C:\Users\<YourUser>\AppData\Roaming\Adobe\UXP\Plugins`.
3. **Launch**:
   - Restart Photoshop.
   - Go to `Plugins` -> `ComfyPanel` to open the control center.

## 中文 - 使用说明

ComfyPanel 是一款专为设计师打造的顶级 AI 协同插件。通过创新的适配器架构，它将复杂的节点式 AI 工作流转化为 Photoshop 中原生、流滑且直觉的参数面板，同时支持本地端与云端算力的无缝调度。

### 🌟 核心功能

- **多引擎支持**：支持 Local ComfyUI，Google NanoBanana, 高性能云端节点（如 BizyAir, RunningHub）一键切换。
- **专业应用管理**：
  - **高度提炼**：将繁琐的节点逻辑抽象为设计师熟悉的 UI 控件，降低上手门槛。
  - **多账号与同步**：灵活管理多个 API 密钥，登录即自动获取云端私有 App 对象。
- **深度画布联动**：
  - **实时感应**：全自动同步图层显隐、像素选区及变形操作。
  - **智能锁定**：支持 LoadImage 节点预览锁定，方便独立管理不同的参考图。
  - **极致导入**：生成结果一键置入画布或蒙版，无延迟、无黑屏，完美融入设计流程。
- **原生 UI 体验**：极简主义设计，提供媲美 PS 原生工具的高水准交互。

### 💻 系统要求

- **Photoshop 版本**：Adobe Photoshop 2025 (v26.0) 或更高版本。

### 🛠️ 安装方式

1. **获取插件**：获取 `ComfyPanel` 源代码文件夹或 `.ccx` 安装包。
2. **安装方式**：
   - **自动安装 (推荐)**：直接双击 `comfypanel.ccx` 运行（需 Adobe Creative Cloud 桌面应用）。
   - **手动安装**：
     - **macOS**: 将 `photoshop_plugin/ComfyPanel` 文件夹复制到 `~/Library/Application Support/Adobe/UXP/Plugins` 目录下。
     - **Windows**: 将文件夹复制到 `C:\Users\<用户名>\AppData\Roaming\Adobe\UXP\Plugins` 目录下。
3. **启用插件**：
   - 重新启动 Photoshop。
   - 在菜单栏选择 `Plugins` -> `ComfyPanel` 即可开启你的创意引擎。
